# HTML Entity Cheatsheet

HTML/XML에서 특수 문자를 안전하게 표현하기 위한 엔티티 참조.

## 필수 엔티티 (HTML에서 이스케이프 필수)

| Char | Entity Name | Entity Number | 설명 |
|------|-------------|---------------|------|
| `&` | `&amp;` | `&#38;` | Ampersand |
| `<` | `&lt;` | `&#60;` | Less than |
| `>` | `&gt;` | `&#62;` | Greater than |
| `"` | `&quot;` | `&#34;` | Double quote |
| `'` | `&apos;` | `&#39;` | Apostrophe |

## 자주 쓰는 엔티티

| Char | Entity Name | Entity Number | 설명 |
|------|-------------|---------------|------|
| ` ` | `&nbsp;` | `&#160;` | Non-breaking space |
| `©` | `&copy;` | `&#169;` | Copyright |
| `®` | `&reg;` | `&#174;` | Registered trademark |
| `°` | `&deg;` | `&#176;` | Degree |
| `±` | `&plusmn;` | `&#177;` | Plus/minus |
| `×` | `&times;` | `&#215;` | Multiplication |
| `÷` | `&divide;` | `&#247;` | Division |
| `€` | `&euro;` | `&#8364;` | Euro |
| `£` | `&pound;` | `&#163;` | Pound |
| `¥` | `&yen;` | `&#165;` | Yen |
| `¢` | `&cent;` | `&#162;` | Cent |

## 따옴표 & 대시

| Char | Entity Name | Entity Number | 설명 |
|------|-------------|---------------|------|
| `'` | `&lsquo;` | `&#8216;` | Left single quote |
| `'` | `&rsquo;` | `&#8217;` | Right single quote |
| `"` | `&ldquo;` | `&#8220;` | Left double quote |
| `"` | `&rdquo;` | `&#8221;` | Right double quote |
| `–` | `&ndash;` | `&#8211;` | En dash |
| `—` | `&mdash;` | `&#8212;` | Em dash |
| `…` | `&hellip;` | `&#8230;` | Ellipsis |

## 수학 기호

| Char | Entity Name | Entity Number | 설명 |
|------|-------------|---------------|------|
| `¼` | `&frac14;` | `&#188;` | 1/4 |
| `½` | `&frac12;` | `&#189;` | 1/2 |
| `¾` | `&frac34;` | `&#190;` | 3/4 |
| `∞` | `&infin;` | `&#8734;` | Infinity |
| `≤` | `&le;` | `&#8804;` | Less than or equal |
| `≥` | `&ge;` | `&#8805;` | Greater than or equal |
| `≠` | `&ne;` | `&#8800;` | Not equal |

## 특수 문자

| Char | Entity Name | Entity Number | 설명 |
|------|-------------|---------------|------|
| `#` | `&num;` | `&#35;` | Number sign |
| `$` | `&dollar;` | `&#36;` | Dollar |
| `%` | `&percnt;` | `&#37;` | Percent |
| `@` | `&commat;` | `&#64;` | At sign |
| `\|` | `&verbar;` | `&#124;` | Vertical bar |
| `~` | `&tilde;` | `&#126;` | Tilde |
| `§` | `&sect;` | `&#167;` | Section |
| `µ` | `&micro;` | `&#181;` | Micro |
