# Toolbox

개인 업무 유틸리티(스크립트, 코드 스니펫, TIL 메모)를 한 곳에 모아 관리하는 저장소입니다.

## 구조

| 폴더 | 설명 |
|---|---|
| `scripts/` | 실행 가능한 유틸리티 스크립트 (서버 관리, 파일 정리 등) |
| `snippets/` | 코드 스니펫, 템플릿 (docker-compose, boilerplate 등) |
| `til/` | Today I Learned — 빠른 메모, 날것의 기록 |

## 사용법

```bash
# 스크립트 실행
bash scripts/<script-name>.sh

# 스니펫 참조
cat snippets/<snippet-name>
```

## TIL vs 블로그

- **TIL**: 날것의 빠른 메모. 형식에 구애받지 않고 바로 기록
- **블로그**: 정리된 글. TIL에서 발전시켜 포스팅
